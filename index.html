Como é uma estrutura básica de um HTML?

(Voltar)

A declaração indica que este documento segue o padrão HTML5. O elemento html é o elemento raiz de uma página HTML, que envolve todo o conteúdo da página. Dentro dele, temos o elemento head, responsável por conter informações meta sobre a página, como configurações e links para estilos ou scripts. O elemento titledefine o título da página, que é exibido na barra de título do navegador ou na aba onde a página está aberta. Já o elemento body representa o corpo do documento e funciona como um contêiner para todo o conteúdo visível, incluindo cabeçalhos, parágrafos, imagens, links, tabelas e listas.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

As listas ordenadas são usadas quando a ordem dos itens é importante, como em etapas de um processo:

O input type="submit"define um botão para enviar os dados do formulário para um manipulador de formulários. Como usar Tabelas

(Voltar)

Para criar uma tabela em HTML, utiliza-se a tag table, que serve como contêiner principal da tabela. Dentro dela, as linhas são definidas pela tag tr (table row), e cada linha pode conter células de cabeçalho th ou células de dados td. As células de cabeçalho geralmente aparecem na primeira linha da tabela e são usadas para identificar o conteúdo das colunas. As células de dados são usadas nas linhas seguintes para exibir as informações propriamente ditas. É possível personalizar a aparência da tabela com atributos HTML ou, preferencialmente, com CSS.

Como inserir videos

(Voltar)

Para inserir vídeos diretamente em uma página HTML, usamos a tag iframe, que permite incorporar conteúdos de outras fontes. Essa tag precisa apontar para o link de incorporação do vídeo (e não o link normal da página do YouTube).
